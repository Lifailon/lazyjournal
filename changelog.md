
# Release 

# New Features

- **HTTP Response Status Coloring:** Added coloring for HTTP response status codes to improve log visibility (, , ).
- **JSON Coloring:** Introduced JSON coloring for enhanced readability ().
- **Settings Enhancements:** Added new settings for color, disable color actions, and startup parameter for filter by date (, ).

- **HTTP Method/Path Coloring Updates:** Improved coloring for HTTP methods and paths, including root directory and keywords (, , , , ).
- **Hotkey Updates:** Enhanced hotkey bindings for filtering and color mode ().
- **Kubernetes:** Added connection check for Kubernetes cluster ().
- **Bat Mode and Bin Check:** Added bat mode and a check for binary files ().
- **Other UI/UX Enhancements:** Updated interface screenshots, flags, keywords coloring, and log file coloring (, , , , , , ).
