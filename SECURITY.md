# Security Policy

If you've discovered a security vulnerability, please do not file a public ticket.

Instead, use the private vulnerability reporting feature provided by GitHub:

- Go to the [Security and Quality](https://github.com/Lifailon/lazyjournal/security) tab.
- Click the **Report Vulnerability** button to open a private security advisory draft.

We will review the vulnerability and keep you updated on the fix. Thank you for helping keep this project secure!
